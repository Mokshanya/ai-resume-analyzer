🚀 Smart Resume Analyzer

An offline-first, heuristically-driven ATS Resume Analyzer capable of instant document transformation for Corporate and Medical profiles.

View Live Application

✨ Features

  • Zero-API Local Processing: Powered entirely by a custom Python regex heuristic engine, requiring zero network calls or paid API keys.
  • Modern Format Support: Seamlessly parses both legacy .pdf and modern .docx files utilizing PyPDF2 and python-docx.
  • Interactive Document Previews: Features an immersive, dynamic CSS-driven UI that physically injects optimized rewrites straight into virtual representations of the user's uploaded document.
  • Intelligent Highlighting: Automatically hunts down weak phrases and violently replaces them with highlighted, industry-standard Action Verbs.

💻 Installation

# Clone the project repository
git clone https://github.com/arjunmehta/Smart-Resume-Analyzer.git

# Install requirements
pip install -r requirements.txt

# Boot the offline server
python app.py
